# Pen-Plotter
Schematics, electronic components and open-source code used in Pen Plotter for my BSc Thesis.

Open-source software used in this project:

Inkscape vector graphics editing software: 
https://inkscape.org/release/inkscape-1.4/  

Inkscape extension for generating G-Code files: 
https://github.com/JTechPhotonics/J-Tech-Photonics-Laser-Tool 

G-Code parser and CNC controller: 
https://github.com/grbl/grbl 

Graphical application for motion controller operation: 
https://github.com/winder/Universal-G-Code-Sender 

IDE environment for microcontroller board programming: 
https://www.arduino.cc/en/software/ 

KiCad-files folder contains KiCad 9.0 project files used to create Pen Plotter schematics (SCH).

Hardware-list.txt contains list of main hardware components used in this project.

STL-files folder contains .stl files of mechanical parts designed for this project.
